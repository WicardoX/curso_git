Introducción a Git

Comandos Básicos

  • git init: inicializa un repositorios
  • git status: nuestra el estado de los archivos del proyecto
  • git add: Agrega los archivos al Area de preparado (Staing Area)
  • git commit -m "": Guarda los archivos preparados en el repositorio
  • git log: Muestras los commits
  • git diff; Muestra las differencias entre los archivos
  • git checkout: permite moverse entre el historial de versiones
  • git config --global user.email "correo@host.com"
  • git config --global user.name "name"

    • Backtracking

  • git checkout HEAD file name: descarta cambios en el area de trabajo
  • git reset HEAD file name: descarta cambios de preparados

    • Branching

  • git branch:
  • git branch branch_name:
  • git checkout branch_name:
  • git merge branch_name:
  • git branch -d branch_name
