Introducción a Git

Comandos Básicos

  • git init: inicializa un repositorio
  • git status: nuestra el estado de los archivos del proyecto
  • git add: Agrega los archivos al Area de preparado (Staing Area)
  • git commit -m "": Guarda los archivos preparados en el repositorio
  • git log: Muestras los commits
  • git diff; Muestra las differencias entre los archivos
  • git checkout
  • git config --global user.email "correo@host.com"
  • git config --global user.name "name"
